# weblocal.es

Landing base en HTML, CSS y JavaScript para presentarte como creador de webs para pequeños comercios y autónomos.

## Archivos
- `index.html` → estructura principal
- `styles.css` → diseño visual y responsive
- `script.js` → menú móvil, animaciones suaves y año automático

## Qué deberías cambiar tú primero
1. En `index.html`, sustituye:
   - `hola@weblocal.es`
   - `https://wa.me/34600000000`

2. Cambia los textos de ejemplos por trabajos reales cuando los tengas.

3. Si quieres añadir tu foto o capturas reales, se puede crear una carpeta `assets/images`.

## Cómo probarla en local
Abre `index.html` con doble clic o con VS Code + Live Server.

## Cómo subirla
Puedes subir esta carpeta a:
- Netlify
- GitHub Pages
- SiteGround
- Hostinger

## Idea recomendada
Cuando quieras, el siguiente paso sería:
- añadir una sección con precios orientativos
- añadir testimonios
- crear una página específica para cada sector (restaurantes, clínicas, peluquerías)
